Fix Wi-Fi, Bluetooth & Network Issues

1. Wi-Fi Problems

Can't Connect to Wi-Fi:

  1. Turn Wi-Fi off and on from menu bar
  2. Restart your router (unplug 30 seconds)
  3. Restart your Mac
  4. Forget network and reconnect:
    • System Settings ‚Üí Wi-Fi
    • Click (i) next to network name
    • "Forget This Network"
    • Reconnect with password
  5. Check router is broadcasting SSID (network name visible)
  6. Try connecting to different network to isolate issue

Connected But No Internet:

  1. Renew DHCP Lease:
    • System Settings ‚Üí Network ‚Üí Wi-Fi ‚Üí Details
    • TCP/IP tab ‚Üí "Renew DHCP Lease"
  2. Change DNS servers:
    • Same location as above ‚Üí DNS tab
    • Add: 8.8.8.8 and 8.8.4.4 (Google DNS)
    • Or: 1.1.1.1 and 1.0.0.1 (Cloudflare)
  3. Reset network settings
  4. Check if router has internet (test other devices)

Slow Wi-Fi:

  • Move closer to router
  • Check for interference (microwaves, cordless phones)
  • Change Wi-Fi channel on router (use 1, 6, or 11 on 2.4GHz)
  • Use 5GHz network if available (faster, shorter range)
  • Run speed test: fast.com
  • Close bandwidth-heavy apps

2. Bluetooth Issues

Bluetooth Won't Turn On:

  1. Restart Mac
  2. Reset Bluetooth module:
    • Hold Shift + Option
    • Click Bluetooth in menu bar
    • "Reset the Bluetooth module"
    • Restart Mac
  3. Delete Bluetooth .plist:
    • /Library/Preferences/com.apple.Bluetooth.plist
    • Restart
  4. Reset SMC and NVRAM

Device Won't Pair:

  1. Make device discoverable (check device manual)
  2. Forget device and re-pair:
    • System Settings ‚Üí Bluetooth
    • Click (i) next to device
    • "Forget This Device"
    • Pair again
  3. Charge Bluetooth device fully
  4. Move devices closer (within 30 feet)
  5. Turn off other Bluetooth devices (reduce interference)
  6. Reset device (check manufacturer instructions)

Audio Stuttering/Cutting Out:

  • Move closer to Mac
  • Remove obstacles between devices
  • Turn off Wi-Fi temporarily (test if interference)
  • Unpair and re-pair device
  • Check battery level

3. AirDrop Not Working

Can't See Other Devices:

  1. Enable AirDrop on both devices:
    • Finder ‚Üí AirDrop (sidebar)
    • Set "Allow me to be discovered by: Everyone"
  2. Enable Wi-Fi and Bluetooth on both devices
  3. Turn off Personal Hotspot
  4. Disable VPN temporarily
  5. Sign in to same iCloud account (for "Contacts Only")
  6. Restart both devices
  7. Toggle AirDrop off/on

Transfer Failing:

  • Move devices closer (within 30 feet)
  • Disable Do Not Disturb
  • Check storage space on receiving device
  • Try sending smaller file first

4. External Display Issues

Display Not Detected:

  1. Check cable connections (both ends)
  2. Try different cable
  3. Try different port on Mac
  4. Restart Mac with display connected
  5. Detect displays manually:
    • System Settings ‚Üí Displays
    • Hold Option key
    • Click "Detect Displays" button
  6. Reset NVRAM
  7. Update macOS

Wrong Resolution:

  • System Settings ‚Üí Displays
  • Select external display
  • Choose "Scaled" and select proper resolution
  • Hold Option and click "Scaled" for more options

5. USB Devices Not Recognized

Device Not Showing Up:

  1. Try different USB-C port
  2. Try different cable
  3. Test device on another computer
  4. Restart Mac with device connected
  5. Check if device needs drivers
  6. Reset SMC
  7. Check System Settings ‚Üí Privacy & Security for blocked devices

6. iCloud Sync Issues

Files Not Syncing:

  1. Check internet connection
  2. Check iCloud status: apple.com/support/systemstatus
  3. Verify iCloud storage not full
  4. Sign out and back in to iCloud
  5. Toggle iCloud Drive off/on:
    • System Settings ‚Üí Apple ID ‚Üí iCloud
    • Turn iCloud Drive off, wait 1 min, turn on

7. Ethernet Connection

Wired Connection Not Working:

  1. Check cable is fully plugged in
  2. Try different ethernet cable
  3. Try different port on router/switch
  4. Check link light on ethernet adapter
  5. System Settings ‚Üí Network ‚Üí Make sure ethernet is at top of list (drag to reorder)
  6. Renew DHCP lease
üí° Network Diagnostics: Use Wireless Diagnostics app (in /System/Library/CoreServices/) to analyze Wi-Fi issues. Hold Option and click Wi-Fi icon for quick signal info.
